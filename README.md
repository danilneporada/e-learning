# E-learning

## Instal Laravel Nova

1. Create Nova directory in project root dir
2. Copy content of nova to this folder

## Install project

```
./vendor/bin/sail up
```

```
sail shell
cp .env.example .env
```

```
sail composer install
```

```
sail artisan migrate
```

## Run tests

Run Php Unit and Feature tests
```
sail test
```
